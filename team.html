The Problem

AI로 영상 만드는 팀이
같이 일할 도구가 없습니다.

지금 AI 제작팀들은 도구 7개를 오갑니다. 생성은 Kling, 협업은 Slack, 리뷰는 Frame.io, 납품은 zip 파일. 그 사이에서 새는 권리·승인·일정은 결국 사람이 떠안습니다.

Kling · Runway · Sora
생성은 되지만 협업이 안 됩니다
개인 도구로 설계됐기 때문에 팀원이 같은 샷을 보고 의견을 남기거나, 클라이언트가 리뷰할 방법이 없습니다.
Frame.io · Wipster
협업은 되지만 AI를 모릅니다
전통 영상 워크플로우 기반이라 프롬프트, 모델, 시드 같은 AI 컨텍스트를 다루지 못합니다.
Notion · Asana
영상 데이터를 다루지 못합니다
체크리스트는 되지만 컷·버전·생성 이력을 미디어 메타와 묶어서 보여주지 못합니다.
엑셀 · 개인 폴더
납품 직전이 가장 위험합니다
모델 권리·승인 이력·자산 리스트를 사람이 매번 모아 만듭니다. 여기서 사고가 납니다.
결과: 5명짜리 제작팀이 도구 7개를 오가며 일합니다.
그 사이에서 시간·권리·신뢰가 새어나갑니다.
The Solution

SOAVIZ는 이 모든 걸
샷 단위로 묶었습니다.

AI 영상 제작의 모든 단계 — 기획·생성·리뷰·승인·납품 — 을 샷이라는 하나의 단위로 묶고, 팀이 같은 화면에서 일하게 만듭니다.

🎬
Shot-based Workbench
모든 영상이 컷 단위로 정리됩니다. Planning → Production → In Review → Approved → Delivered. 각 샷은 자기만의 프롬프트, 버전, 댓글, 권리 로그를 가집니다.
  • 5단계 칸반 보드 + 테이블 + 갤러리 뷰
  • 한 샷에 N개 버전 누적 (롤백·비교 가능)
  • 드래그앤드롭으로 상태 전이
🔌
Multi-AI BYOK
Kling, Runway, OpenAI Sora, ElevenLabs를 SOAVIZ 안에서 호출합니다. 본인 API Key를 연결하므로 SOAVIZ는 마진을 안 가져갑니다.
  • Personal API Vault (개인) / Workspace API Vault (팀)
  • 모델별 사용량·비용 자동 기록
  • 키 마스킹·암호화·팀원 비공개
🔗
Client Review Link
클라이언트가 별도 계정 없이 영상을 보고 승인·수정 요청할 수 있는 링크를 생성합니다. 모든 피드백은 SOAVIZ로 자동 동기화되고 Approval History에 남습니다.
  • 씬·시간 단위 댓글
  • 승인 / 수정 자동 분류
  • 만료·접근 제어 가능
📦
Delivery Package
납품 직전에 가장 사고가 많이 나는 단계를 자동화합니다. Approved 샷을 한 클릭으로 영상·권리·승인 로그·메타데이터까지 묶어 패키지로 만듭니다.
  • Final Video / Thumbnails / Subtitles
  • Prompt Log / Model Usage Log
  • Rights Report / Approval History / Asset List
How it works

기획부터 납품까지, 4단계.

STEP 1
프로젝트 생성
개인 또는 팀 프로젝트로 시작. 언제든 한 클릭으로 팀으로 전환할 수 있습니다.
STEP 2
샷 분해 + 생성
Brief를 AI가 자동으로 샷 단위로 나눕니다. 각 샷에서 Kling·Runway·Sora를 직접 호출.
STEP 3
리뷰 + 승인
팀원·클라이언트가 같은 샷을 보고 댓글·수정 요청·승인. 모든 흐름이 자동 기록됩니다.
STEP 4
납품 패키지
Approved 샷을 영상·권리 로그·승인 이력까지 묶어 한 번에 패키지화·납품.

FAQ

자주 묻는 질문

SOAVIZ는 어떤 AI 모델을 쓸 수 있나요?
Kling, Runway Gen-3, OpenAI Sora, ElevenLabs, Stability AI 등 주요 영상·음성 생성 API를 지원합니다. SOAVIZ는 BYOK(Bring Your Own Key) 구조로, 본인 API Key를 연결해서 사용합니다. SOAVIZ는 API 비용에 마진을 붙이지 않습니다.
기존 도구(Frame.io, Notion)와 어떻게 다른가요?
기존 도구는 일반 영상이나 일반 프로젝트용입니다. SOAVIZ는 AI 영상 제작 워크플로우 — 프롬프트, 모델, 시드, 권리 로그 — 를 처음부터 1급 시민으로 다룹니다. Frame.io에 없는 것: AI 컨텍스트. Notion에 없는 것: 미디어 메타. SOAVIZ에 둘 다 있습니다.
개인이 쓰다가 팀으로 옮기면 데이터는 어떻게 되나요?
그대로 유지됩니다. 같은 프로젝트를 한 클릭으로 팀 프로젝트로 전환하면, 기존 샷·버전·댓글·권리 로그가 모두 그대로 살아남고, 위에 멤버·리뷰·승인·납품 기능이 활성화됩니다.
Rights Report는 법적 효력이 있나요?
Rights Report는 SOAVIZ가 사용된 모델·라이선스·프롬프트·승인 흐름을 자동으로 기록한 문서입니다. 법적 효력은 관할권에 따라 다르지만, 클라이언트·법무팀·감사 대응 시 사실 기반 자료로 사용할 수 있습니다. Business 플랜 이상에서 PDF로 export 가능합니다.
베타에 참여하면 어떤 혜택이 있나요?
베타 사용자는 정식 출시 후 6개월간 Studio 플랜을 50% 할인된 가격으로 사용할 수 있습니다. 제품 로드맵에 직접 영향을 줄 수 있는 정기 라운드테이블에도 초대됩니다.
데이터는 어디에 저장되나요?
프로젝트 메타데이터는 SOAVIZ의 보안 인프라에, 영상 파일은 사용자가 지정한 스토리지(자체 S3, GCS 등)에 저장 가능합니다. Enterprise 플랜은 전용 인스턴스 옵션을 제공합니다.
