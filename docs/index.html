영상 제작 전용 AI OS · v0.3 beta

⚡ 빠른 시작

처음 사용하는 경우 5단계로 첫 번째 샷을 생성할 수 있습니다.

  • 설정 → API 키 등록
    좌측 사이드바 ⚙ 설정에서 사용할 AI 서비스 키를 입력합니다. 키는 브라우저에 AES-GCM 암호화로 저장되며 외부로 전송되지 않습니다.
  • 새 프로젝트 생성
    파이프라인 탭 → + 새 프로젝트. 포맷(시리즈/영화/숏폼/MV/광고)과 장르를 선택합니다.
  • 에피소드 → 씬 → 샷 구조 만들기
    프로젝트 안에서 에피소드 → 씬 → 샷 계층으로 스토리를 구조화합니다.
  • 샷 스튜디오에서 이미지/영상 생성
    샷을 선택하고 ✦ 샷 스튜디오에서 프롬프트를 작성한 뒤 생성합니다. 결과물은 자동으로 라이브러리에 저장됩니다.
  • 승인 및 내보내기
    비교 모드에서 최적의 에셋을 선택 → approved 상태로 변경 → 프로젝트 패키지 내보내기.

⚙ 초기 설정

⚙ 설정 페이지에서 각 서비스의 API 키를 등록합니다. BYOK(Bring Your Own Key) 방식으로 서버에는 키가 저장되지 않습니다.

▦ 파이프라인

프로젝트 → 에피소드 → 씬 → 샷의 4계층 구조로 콘텐츠를 관리합니다.

프로젝트

포맷(short/series/film/mv/ad), 장르(복수 선택), 색상 태그, 마감일, 스타일 레퍼런스를 설정합니다.

에피소드

시리즈 포맷에서는 에피소드 단위로 분리합니다. 영화/숏폼은 에피소드 1개로 운영합니다.

씬

씬 헤딩(INT. CAFE — DAY), 핵심 비트, 등장인물 태그를 기록합니다.

샷

샷 타입(WS/MS/CU/ECU/OTS/POV), 카메라 무브먼트, 이미지/영상 프롬프트, 모델을 지정합니다.

◎ 캐릭터

프로젝트 전체에서 재사용 가능한 캐릭터 페르소나를 관리합니다.

캐릭터 생성

이름, 역할(주인공/조연/빌런), 연령대, 성격 프리셋(50종), 말투 톤 프리셋(50종)을 설정합니다.

키이미지 스튜디오

캐릭터 카드 → 키이미지 스튜디오 버튼을 클릭하면 외형 설명으로 기준 이미지 4종(정면/측면/후면/전신)을 생성합니다.

  1. 외형 메모 작성
    나이, 체형, 머리색, 눈색, 특징적 의상 등 자연어로 기술합니다.
  2. 비주얼 프롬프트 생성
    AI가 외형 메모를 fal.ai용 최적화 프롬프트로 변환합니다.
  3. 이미지 생성 및 저장
    각 앵글별 이미지를 생성 → 마음에 드는 컷을 캐릭터 기준 이미지로 저장합니다.

보이스 연결

캐릭터에 ElevenLabs Voice ID를 연결하면 보이스 스튜디오에서 해당 캐릭터의 목소리로 대사를 생성할 수 있습니다.

◉ 스타일 & 시네마

프로젝트의 시각적 언어를 정의하는 룩북을 구성합니다.

스코프설명
팔레트색상 코드 배열, 전체 색감 방향성
라이팅조명 스타일 설명 (high-key, noir, natural 등)
카메라렌즈·무브먼트·초점 스타일 기술
음악배경음악 장르·템포·분위기 메모
의상시대적·장르적 의상 방향성
전체위 항목 통합 레퍼런스 문서

시네마 레퍼런스 — 참조 영화/작품 목록(제목·년도·참조 요소)을 등록하면 샷 프롬프트 생성 시 자동으로 스타일 힌트가 적용됩니다.

✦ 샷 스튜디오

이미지·영상·업스케일·립싱크 생성의 핵심 작업 공간입니다.

Auto-Archive

생성된 모든 에셋은 자동으로 라이브러리에 저장됩니다. 샷-에셋 연결은 끊기지 않으며, candidate → approved → final 상태 전환으로 작업 흐름을 관리합니다.

비교 생성 (Compare Group)

동일 프롬프트로 A/B/C/D/E 최대 5개 변형을 동시에 생성합니다. 비교 모드에서 나란히 검토 후 최적안을 approved로 승인합니다.

지원 모델

현재 지원: Kling 2.1, Runway Gen-3 Alpha, fal.ai Flux 계열, SDXL, Stable Video Diffusion

♪ 보이스 스튜디오

캐릭터 목소리로 대사를 생성하거나 커스텀 보이스를 관리합니다.

TTS 생성

대사 텍스트 입력 → 캐릭터 선택(ElevenLabs Voice 연결 필요) → stability/similarity/style 파라미터 조정 → 생성

보이스 라이브러리

ElevenLabs에서 공유된 보이스를 검색하거나 커스텀 클론 보이스를 캐릭터에 연결합니다.

♫ 뮤직 & SFX

씬 분위기에 맞는 배경음악과 효과음을 생성합니다.

뮤직 생성 (Suno AI)

장르·분위기·가사 여부·길이를 설정하고 Suno API로 생성합니다. 생성물은 라이브러리 music 타입으로 저장됩니다.

SFX 생성

씬 설명을 기반으로 효과음을 생성합니다. 발걸음, 문 소리, 환경음 등 자연어 설명으로 생성 가능합니다.

◫ 라이브러리

프로젝트 전체 에셋을 타입·상태·날짜 기준으로 필터링하고 관리합니다.

에셋 타입설명
image이미지 생성 결과물
video영상 생성 / 립싱크 결과물
audioTTS / SFX / 음악
upscale업스케일된 고해상도 이미지
textAI 생성 스크립트·대사

에셋 상태: queued → generating → ready → candidate → approved → final

⊞ 비교 모드

동일 샷의 A/B/C 변형 에셋을 나란히 놓고 최적안을 선택합니다.

  1. 샷 스튜디오에서 compare group 생성
    프롬프트 동일, 시드/파라미터 변형으로 다수 생성
  2. 비교 모드 탭 열기
    compare group ID를 선택하면 A/B/C 슬롯에 에셋이 자동 배치됩니다
  3. 승인
    최적안 선택 → Approve → 샷의 approved_asset_id에 연결됩니다

💬 FAQ

API 키가 안전한가요?
네. 키는 브라우저의 Web Crypto API로 AES-GCM 256비트 암호화된 후 localStorage에만 저장됩니다. 서버로는 전송되지 않습니다.
생성된 에셋이 사라졌어요.
스토리지 티어가 cold/frozen으로 전환되면 미리보기가 숨겨질 수 있습니다. 라이브러리 탭에서 archived 필터를 해제하고 확인하세요.
프로젝트를 ZIP으로 내보내려면?
파이프라인 → 프로젝트 카드 메뉴 → 내보내기 → 포맷 선택(zip/fcpxml/edl). Pro 플랜 이상에서 사용 가능합니다.
캐릭터를 여러 프로젝트에서 공유할 수 있나요?
캐릭터는 계정 레벨에서 관리됩니다. 생성 시 특정 프로젝트에 귀속하거나 글로벌(모든 프로젝트 사용 가능)로 설정할 수 있습니다.
Sovita는 무엇인가요?
Sovita는 Soaviz Studio와 별개인 페르소나 관리 전용 서비스입니다. Soaviz Studio는 영상 제작에 특화되어 있습니다.
