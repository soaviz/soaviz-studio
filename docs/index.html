서비스 창업스토리

혼자서 영상을 만드는 크리에이터를 위해 만들었습니다.

기획은 노션, 스크립트는 ChatGPT, 음성은 다른 앱, 편집은 또 다른 툴 — 크리에이터 한 명이 하루에 열어야 하는 앱이 너무 많았습니다. 아이디어가 떠올랐을 때 곧장 결과물까지 갈 수 있는 직선 경로가 없었습니다.

"AI가 정리하면, 당신은 창작만 할 수 있다"

soaviz studio는 그 질문에서 시작했습니다. 반복적인 정리·변환·포맷 작업을 AI가 대신하고, 크리에이터는 오직 아이디어와 선택에만 집중할 수 있도록. 기획부터 사운드까지, 흩어진 작업을 하나의 흐름으로 연결하는 것이 목표였습니다.

지금도 매일 업데이트 중입니다.

실제 콘텐츠 제작 현장의 피드백을 바탕으로, 필요한 기능을 빠르게 추가하고 있습니다. AI Pre-production Board, 오디오 에셋 추출, 작업 일기까지 — 모두 실제 크리에이터의 요청에서 나온 기능들입니다.

AI 콘텐츠 크리에이터 은교가 직접 만들고, 직접 씁니다.
시작하기

3분 빠른 시작

처음 접속했다면 이 순서대로 따라해 보세요.

Step 1 — AI 키 등록

  1. 사이드바 하단 설정 클릭
  2. AI Keys 탭에서 OpenAI API Key 입력 후 저장
    sk-proj-... 형식의 키가 필요합니다
  3. ElevenLabs TTS 사용 시 ElevenLabs Key도 함께 입력

API 키는 브라우저 로컬 스토리지에만 저장됩니다. 서버로 전송되지 않습니다.

Step 2 — 첫 번째 프로젝트 만들기

  1. 사이드바 프로젝트 클릭
  2. + 새 프로젝트 버튼 클릭 → 제목 입력
  3. 프로젝트가 생성되면 상단에 프로젝트명이 표시됩니다

Step 3 — 기획/스토리 작성

  1. 사이드바 기획/스토리 클릭
  2. 로그라인, 세계관, 시놉시스 입력
  3. AI 생성 버튼으로 아이디어 초안 자동 생성 가능

Step 4 — 샷 리스트 생성

  1. 사이드바 샷 리스트 클릭
  2. 상단 탭에서 샷리스트 생성기 선택
  3. 기획/스토리 불러오기 또는 직접 시놉시스 입력
  4. 디테일 레벨 선택: Basic / Director / AI Production
  5. 샷리스트 생성 버튼 클릭 → Shot Card 자동 생성

AI Production 모드를 권장합니다. 각 샷마다 Midjourney용 Image Prompt, Kling/Runway용 Video Prompt가 자동으로 생성됩니다.

시작하기

전체 워크플로우

soaviz studio의 단계별 제작 흐름을 한눈에 파악하세요.

기획 단계

기획/스토리에서 세계관·로그라인·시놉시스를 작성합니다. AI 보조로 초안을 빠르게 잡고, 프로젝트에서 일정과 마감을 관리합니다.

프리프로덕션 단계

샷 리스트(AI Pre-production Board)에서 씬별 Shot Card를 생성합니다. 캐릭터 페르소나로 등장인물을 정의하고, 룩북 & 레퍼런스로 비주얼 방향성을 확정합니다.

영상 생성 단계

Shot Card의 Image Prompt를 Midjourney에 붙여넣어 씬 이미지를 만들고, Video Prompt를 Kling·Runway·Sora에 활용합니다. 영상 편집에서 클립을 조합·컷편집합니다.

사운드 제작 단계

내레이션/TTS로 스크립트를 음성으로 변환하고, 음악 제작으로 BGM을 만듭니다. 효과음으로 사운드 레이어를 추가하고, 오디오 에셋으로 참고 영상에서 소리를 추출해 활용합니다.

후반/패키징 단계

결과 보관에서 생성된 이미지·영상·오디오를 프로젝트별로 저장합니다. 최종 출력에서 완성된 작품을 아카이브하고 내보냅니다.

Today — 오늘

매일 제작 흐름을 기록하는 작업 일기 + 대시보드

화면 구성

  • 오늘의 날씨/컨디션: 제작 시작 전 현재 상태 기록
  • 오늘의 목표: 당일 완료할 작업 목록
  • 작업 일기: 진행 상황, 메모, 결정 사항 자유 기록
  • AI 피드백: 입력한 내용 기반으로 다음 작업 제안

How to 사용법

  1. 매일 접속 시 Today 화면이 자동으로 열립니다
  2. 오늘의 목표에 할 일을 입력하고 체크박스로 완료 표시
  3. 작업 중 메모·결정 사항을 작업 일기에 기록
  4. 하루 마감 시 저장 클릭 → 히스토리에 자동 누적

과거 날짜를 클릭하면 이전 작업 일기를 다시 열람할 수 있습니다. 프로젝트 회고에 활용하세요.

기획

기획/스토리

세계관·로그라인·시놉시스를 정리하고 AI 보조로 스토리를 발전시키는 공간

주요 기능

  • 로그라인: 1~2문장으로 작품의 핵심 요약
  • 세계관 설정: 배경, 시대, 규칙, 분위기 정의
  • 시놉시스: 줄거리 전개 정리 (에피소드 단위 가능)
  • 키워드 태그: 장르·분위기·키워드로 작품 분류
  • AI 확장: 입력한 설정 기반으로 추가 아이디어 생성

How to — 세계관 작성

  1. 기획/스토리 탭 클릭
  2. 새 기획 만들기 또는 기존 기획 선택
  3. 로그라인 → 세계관 → 시놉시스 순서로 입력
  4. AI 확장 생성 버튼으로 씬 아이디어, 갈등 구조, 반전 요소 자동 제안 받기
  5. 완성된 기획은 샷 리스트 생성기에서 기획 불러오기로 연결됩니다

How to — 샷 리스트와 연동

기획/스토리에서 작성한 시놉시스는 샷 리스트 생성기의 기획 불러오기 버튼을 통해 자동으로 입력됩니다. 매번 다시 타이핑할 필요가 없습니다.

기획

프로젝트

영상 작품 단위로 제작 일정과 진행 상태를 관리

프로젝트 구조

  • 프로젝트명: 작품 제목 또는 시리즈명
  • 시작일 / 마감일: 인라인 캘린더 팝업으로 날짜 선택
  • 진행 상태: 기획중 / 제작중 / 편집중 / 완료
  • 메모: 프로젝트별 자유 메모

How to — 프로젝트 생성

  1. 사이드바 프로젝트 클릭
  2. + 새 프로젝트 클릭 → 제목 입력
  3. 시작일 셀 클릭 → 팝업 캘린더에서 날짜 선택
  4. 마감일 설정 → 상태 드롭다운에서 현재 단계 선택
  5. 상단 프로젝트 셀렉터에서 활성 프로젝트 전환 가능

활성 프로젝트를 설정하면 Today, 샷 리스트, 결과 보관 등 모든 화면이 해당 프로젝트 기준으로 연동됩니다.

샷 리스트 v3 · AI Pre-production Board

씬별 Shot Card 자동 생성 + Midjourney·Kling·Runway 프롬프트 즉시 복사

3단계 디테일 모드

Basic

씬 번호·설명·카메라 기본 7개 필드. 빠른 초안용.

조명·분위기·감독 의도 추가 10개 필드. 연출 중심.

AI Production ★

14필드 풀스펙. Image·Video 프롬프트 자동 생성. 추천.

How to — 샷 리스트 생성

  1. 사이드바 샷 리스트 클릭 → 샷리스트 생성기 탭 선택
  2. 기획 불러오기 버튼 클릭 → 기획/스토리에서 작성한 시놉시스 자동 입력
    (또는 시놉시스 텍스트 직접 입력)
  3. 목적 선택: Midjourney / Kling / Runway / 스토리보드
  4. 디테일 레벨: AI Production 선택 (권장)
  5. 씬 수 설정 후 샷리스트 생성 클릭
  6. 생성된 Shot Card 검토 → 필요 시 개별 수정

How to — 프롬프트 복사

각 Shot Card 하단에 3개의 복사 버튼이 있습니다:

  • 📷 Image Prompt 복사: Midjourney에 바로 붙여넣기
  • 🎬 Video Prompt 복사: Kling / Runway / Sora에 바로 붙여넣기
  • 📋 Full Card 복사: 14개 필드 전체를 텍스트로 복사

How to — 샷리스트 편집

  1. 생성된 Shot Card의 각 필드를 직접 클릭해 수정 가능
  2. 씬 추가: 목록 하단 + 버튼 또는 카드 사이에 삽입
  3. 씬 삭제: 카드 우상단 × 버튼
  4. 순서 변경: 드래그 앤 드롭
  5. 완성된 샷리스트는 내보내기로 텍스트 파일 저장

AI Production 모드는 씬당 더 많은 토큰을 사용합니다. 씬 수가 많을수록 OpenAI API 비용이 증가합니다.

캐릭터 페르소나

① + 새 캐릭터로 인물 등록  →  ② 이름·나이·역할·직업·MBTI 입력  →  ③ AI 프롬프트 생성으로 Midjourney 레퍼런스 제작

캐릭터 카드 구성

  • 이름 · 나이: 카드 상단에 대형으로 표시 (이름 / 나이 배지)
  • 역할: 주인공 / 조력자 / 악당 등 서사 포지션
  • 직업: 캐릭터의 직업·신분 (v3.7 신규 필드)
  • MBTI: 성격 유형 배지로 시각화
  • 외형 묘사: 헤어·얼굴·체형·의상 상세 텍스트
  • 성격 태그: 핵심 성격 키워드 3~5개
  • 배경 스토리: 캐릭터 히스토리 요약
  • AI 이미지 프롬프트: Midjourney 최적화 캐릭터 프롬프트 (영문)

How to — 캐릭터 만들기

  1. 캐릭터 페르소나 클릭 → + 새 캐릭터
  2. 이름·나이·역할·직업 입력, 필요 시 MBTI 선택
  3. 외형 묘사 텍스트 입력 (헤어·체형·의상·분위기)
  4. AI 프롬프트 생성 클릭 → 외형 묘사 기반으로 Midjourney 프롬프트 자동 생성
  5. 생성된 프롬프트를 복사해 Midjourney에서 레퍼런스 이미지 제작 후 카드에 첨부

캐릭터 카드를 먼저 만들어두면 샷리스트 생성 시 등장인물 필드에 자동으로 참조됩니다. 직업 필드는 Pre-Production Engine의 Character Pack 생성 시에도 활용됩니다.

  • 촬영 스타일 메모: 렌즈·필름 그레인·색보정 방향 텍스트
  • Style Prompt Fragment: AI 이미지·영상 생성 시 일괄 적용되는 스타일 서픽스
  • 출처: 직접 입력 또는 Cinema Library에서 추출된 스타일 표시

How to — 직접 Style Bible 만들기

  1. 룩북 & 레퍼런스 클릭
  2. + 새 스타일 버튼 → 이름·태그 입력
  3. 분위기 키워드 및 촬영 스타일 메모 입력
  4. Style Prompt 생성으로 키워드 기반 Midjourney 서픽스 자동 생성
  5. 저장된 스타일은 샷리스트 생성 시 자동 반영됨

How to — Cinema Library에서 스타일 추출하기

  1. Cinema Library에서 레퍼런스 영상 등록 후 장면 마킹
  2. 클립의 🔍 스타일 프롬프트 추출 버튼 클릭 → AI가 해당 장면의 시각 스타일을 분석
  3. 추출 완료 시 Style Bible에 자동 저장 — 버튼이 📋 스타일 프롬프트 복사로 전환
  4. 상단 🎨 Style Bible 버튼 또는 변환 배너를 클릭해 저장된 스타일 확인
  5. 저장된 스타일 프롬프트가 샷리스트 생성 시 자동으로 반영됨

Cinema Library에서 추출한 스타일 프롬프트 Fragment는 샷리스트의 Image/Video Prompt에 자동으로 삽입됩니다. 여러 스타일을 등록할수록 씬별 톤 일관성이 높아집니다.

How to — Pre-Production Pipeline 실행

  1. 사이드바 기획/스토리 클릭 → 스토리 또는 시놉시스 작성

  3. SOAVIZ AI가 순서대로 처리합니다:
    ① 캐릭터 팩 생성 (등장인물 자동 추출 + 상세 프로필)
    ② 세계관 바이블 생성 (배경·규칙·분위기 문서화)
    ③ Episode/Scene/Shot 분해 + Shot Card 일괄 생성

Character Pack 상세

스토리에서 등장인물을 자동으로 찾아 각 캐릭터별로 다음 항목을 생성합니다:

  • 이름 · 역할: 주인공/조력자/악당 등 서사 역할 자동 분류
  • 외형 묘사: 헤어·체형·의상·분위기 상세 텍스트
  • 성격 태그: 3~5개 핵심 성격 키워드
  • AI 이미지 프롬프트: Midjourney 최적화 캐릭터 프롬프트 (영문)
  • 모션 프롬프트: Kling/Runway용 동작 묘사 프롬프트

이미 캐릭터 카드가 있으면 재생성을 건너뜁니다. force: true 옵션 없이는 기존 데이터를 덮어쓰지 않습니다.

World Bible 상세

스토리 배경을 분석해 다음 항목으로 구성된 세계관 문서를 자동 생성합니다:

  • 세계관 제목: 배경 공간·시대를 압축한 제목
  • 장르 · 분위기: 사이버펑크 / 판타지 / 근미래 SF 등 태그
  • 핵심 법칙: 이 세계만의 규칙이나 설정 (마법 체계, 기술 수준 등)
  • 주요 장소: 스토리 배경이 되는 공간 목록
  • 시각적 톤: 색감·조명·분위기 방향성
  • 월드 프롬프트: 이 세계관을 AI 이미지로 표현할 때 쓸 배경 서픽스

Shot Pipeline 상세

Character Pack · World Bible 정보를 컨텍스트로 활용해 더 정밀한 샷을 생성합니다:

  • Episode → Scene → Shot 3계층 자동 분해
  • 각 Shot에 담당 캐릭터 자동 할당
  • World Bible의 시각 톤이 Image/Video Prompt에 반영
  • 기존 샷리스트 생성기와 동일한 14필드 AI Production Shot Card 형식

전체 파이프라인은 내부적으로 여러 AI 호출을 순차 실행합니다. 스토리 길이에 따라 10~40초 소요될 수 있습니다. OpenAI API 키가 설정되어 있어야 합니다.

영상 생성

영상 편집

① 영상 업로드  →  ② 편집 버전 선택 (A~F)  →  ③ 영상 제작으로 자동 편집본 생성

지원 기능

  • AI 자동 편집: 영상 분석 후 6가지 편집안을 자동 생성
  • 클립 트리밍: 시작/끝점 드래그로 구간 설정
  • 클립 병합: 여러 영상을 순서대로 이어붙이기
  • 오디오 교체/믹스: 원본 오디오 음소거 후 TTS·음악 오버레이
  • 출력 형식: MP4 (H.264) 기본 출력

편집 버전 6종 (A ~ F)

영상 업로드 후 AI가 자동으로 6가지 편집안을 제안합니다. 빠른 추천 편집 3종과 더 긴 롱폼 버전 3종으로 구성됩니다:

  • A · B · C: 30초 이하 빠른 편집안 — 하이라이트 중심, 비트 컷, 감성 슬로우모션 등 각기 다른 리듬
  • D · E · F: 롱폼 편집안 — 스토리텔링형 / 몰입형 내러티브 / 풀 버전 등 더 긴 호흡의 편집

각 버전 카드에는 평균 점수(avg pt)·최고 점수(top pt)와 편집 리듬·길이 정보가 표시됩니다. 원하는 버전을 선택해 렌더링하세요.

지원 파일 형식

  • 입력: MP4 MOV WEBM AVI — 최대 500MB
  • 출력: MP4

How to — AI 자동 편집

  1. 사이드바 영상 편집 클릭
  2. 파일 추가 클릭 → 영상 파일 선택 (드래그 드롭 가능)
  3. AI가 영상을 분석해 A~F 6가지 편집 버전 자동 생성
  4. 버전 카드를 비교해 원하는 편집안 선택
  5. 영상 제작 버튼 클릭 → 렌더링 시작 (진행 중 로딩 메시지 표시)
  6. 완료 후 MP4 파일 다운로드

렌더링 중 표시

영상 제작 중에는 "컷을 정밀하게 자르는 중…", "트랜지션을 부드럽게 다듬는 중…" 등 30가지 로딩 메시지가 2.6초마다 자동으로 전환됩니다. 앱이 정상 동작 중이라는 시각적 피드백입니다.

브라우저 메모리에서 처리되므로 고화질 대용량 파일은 렌더링 시간이 길어질 수 있습니다. 1080p 기준 분당 약 30초~1분 소요.

오디오 에셋 NEW

영상 파일에서 오디오를 추출해 재사용 가능한 에셋으로 저장

무엇에 쓰나요?

레퍼런스 영상, 촬영 영상, 소셜 영상에서 오디오를 분리해 BGM·효과음·내레이션 소스로 활용합니다. 브라우저 내에서 처리되므로 파일이 서버로 전송되지 않습니다.

지원 입력 형식

  • 파일 업로드: MP4 MOV WEBM — 최대 500MB
  • URL 입력: 직접 접근 가능한 영상 URL (CORS 허용 필요)

출력 형식

  • WAV: 무손실 원본 품질 (기본)
  • MP3: 압축 파일 (용량 절약)

How to — 파일에서 오디오 추출

  1. 사이드바 오디오 에셋 클릭
  2. 파일 선택 클릭 또는 영상 파일을 드래그 앤 드롭
  3. 파일명·길이·용량이 표시되면 확인
  4. 출력 형식 선택 (WAV 권장)
  5. 오디오 추출 버튼 클릭 → 진행바 대기
  6. 추출 완료 → 플레이어에서 미리듣기 가능
  7. 다운로드 클릭 → 로컬 저장

How to — URL에서 오디오 추출

  1. URL 입력 탭 클릭
  2. 영상 파일 URL 붙여넣기 (직접 링크여야 함)
  3. 불러오기 → 파일 정보 확인
  4. 이후 파일 방식과 동일하게 추출

처리는 Web Audio API 기반으로 브라우저 내에서만 이루어집니다. 파일 내용이 외부 서버로 전송되지 않습니다.

저작권이 있는 영상의 오디오 추출은 개인 용도 또는 라이선스 허용 범위 내에서만 사용하세요.

후반/패키징

결과 보관

① 탭 선택 (시나리오·샷리스트·에셋)  →  ② 생성물 저장  →  ③ 씬 번호·태그로 분류해서 관리

탭 구성

  • 시나리오 탭: 완성된 시나리오·스크립트 텍스트 저장 및 열람
  • 샷리스트 탭: 생성·저장된 샷리스트 목록 관리 및 불러오기
  • 에셋 탭: 이미지·영상 클립·오디오 파일 그리드로 관리

저장 가능 에셋 타입

  • 이미지: Midjourney 씬 이미지, 캐릭터 레퍼런스
  • 영상 클립: Kling·Runway 생성 클립
  • 오디오: TTS 내레이션, BGM, SFX, 추출 오디오
  • 시나리오·샷리스트: 텍스트 기반 기획 결과물

How to — 시나리오·샷리스트 보관

  1. 결과 보관 클릭 → 시나리오 또는 샷리스트 탭 선택
  2. 저장된 목록이 자동으로 표시됨 (기획/스토리 페이지·샷리스트 페이지에서 저장한 결과)
  3. 항목 클릭 → 내용 열람 및 복사
  4. 필요 시 내보내기(Export)로 텍스트 파일 저장

How to — 에셋 저장

  1. 에셋 탭 클릭
  2. + 에셋 추가 → 파일 선택 또는 URL 입력
  3. 프로젝트·씬 번호·태그 입력으로 분류
  4. 저장된 에셋은 썸네일 그리드로 확인
  5. 씬 번호 필터로 특정 씬 에셋만 모아보기

샷리스트 탭에서 이전에 저장한 샷리스트를 불러오면 Pipeline 워크플로우를 이어서 진행할 수 있습니다. 시나리오 탭으로 기획 문서를 한 곳에 모아두세요.

자주 묻는 질문

AI 기능을 쓰려면 별도 키가 필요한가요?

네. 설정 → AI Keys에서 OpenAI API Key(GPT-4o 기반 기능)와 ElevenLabs API Key(TTS 기능)를 등록해야 합니다. 키는 브라우저에만 저장되며 서버로 전송되지 않습니다.

AI Production 모드가 너무 오래 걸려요

씬 수가 많을수록 시간이 길어집니다. 14개 필드를 씬마다 생성하므로 10씬 기준 약 20~40초 소요됩니다. 빠른 초안이 필요하면 Basic 모드로 먼저 생성 후 AI Production으로 다시 생성하세요.

오디오 에셋에서 파일이 업로드 안 돼요

지원 형식(MP4, MOV, WEBM)이 맞는지 확인하고, 파일 크기가 500MB 이내인지 확인하세요. 500MB 초과 시 파일을 먼저 분할하거나 압축해야 합니다.

생성한 샷리스트는 어디에 저장되나요?

현재 브라우저 로컬 스토리지에 저장됩니다. 내보내기 버튼으로 텍스트 파일(.txt)을 로컬에 저장해두는 것을 권장합니다.

모바일에서도 사용 가능한가요?

네. 하단 탭 바를 통해 주요 화면을 이동할 수 있습니다. 단, 영상 편집·오디오 추출 등 고성능 기능은 PC 환경을 권장합니다.

Midjourney / Kling / Runway 계정이 별도로 필요한가요?

네. soaviz studio는 프롬프트를 생성·복사하는 역할을 하며, 실제 이미지·영상 생성은 해당 외부 서비스에서 이루어집니다. 각 서비스 계정이 필요합니다.

저작권은 어떻게 되나요?

soaviz studio에서 작성한 기획·샷리스트·텍스트의 저작권은 사용자에게 있습니다. AI 생성 이미지·영상의 저작권은 각 외부 서비스의 약관을 따릅니다.

데이터 백업은 어떻게 하나요?

각 화면의 내보내기 버튼으로 개별 데이터를 저장하세요. 설정 → 데이터 내보내기에서 전체 데이터를 JSON 형식으로 한 번에 백업할 수 있습니다.

문제 해결

AI 생성이 실패해요 / 오류가 나요

  • 설정에서 OpenAI API Key가 올바르게 입력되었는지 확인
  • API 키의 사용량 한도(Quota)가 초과되지 않았는지 OpenAI 대시보드에서 확인
  • 네트워크 연결 상태 확인 후 새로고침
  • 그래도 안 될 경우 API 키를 삭제 후 재입력

TTS 음성이 생성되지 않아요

  • 설정에서 ElevenLabs API Key 입력 여부 확인
  • ElevenLabs 계정의 캐릭터 크레딧 잔량 확인
  • 선택한 보이스 ID가 본인 계정에서 접근 가능한지 확인

영상 편집 렌더링이 실패해요

  • 파일 크기가 500MB 이하인지 확인
  • 지원 형식(MP4, MOV, WEBM)인지 확인
  • Chrome / Edge 최신 버전 사용 권장
  • 브라우저 탭을 닫았다가 다시 열기 (메모리 초기화)

오디오 에셋 추출이 안 돼요

  • 파일 형식·크기 확인 (MP4/MOV/WEBM, 500MB 이하)
  • Chrome 버전 80 이상에서 Web Audio API가 지원됩니다
  • 파일이 DRM 보호된 경우 추출 불가

화면이 갑자기 흰 화면이 돼요

  • ⌘ + Shift + R (강제 새로고침)
  • 그래도 안 될 경우 브라우저 캐시 삭제 후 재접속
