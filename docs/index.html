AI로 영상 만드는 가장 쉬운 방법 — 글, 캐릭터, 목소리, 음악, 영상까지 한 화면에서.

soaviz studio가 도와드리는 일

soaviz studio는 영상 콘텐츠를 만드시는 분들을 위한 작업 공간입니다. 스토리 쓰기, 캐릭터 만들기, 분위기 정하기, 보이스 녹음, 배경음악 깔기, 영상 만들기, 결과 비교하기까지 — 한 편의 영상을 완성하는 데 필요한 11가지 도구가 한 화면 안에 모여 있습니다.

브라우저만 있으면 어디서든 접속 가능하고, 작업물은 자동으로 저장됩니다. 쓰던 작업을 다른 컴퓨터·스마트폰에서도 그대로 이어서 할 수 있어요.

만든 사람의 한 마디

정리를 가장 못하는 사람이, 정리해주는 앱을 만들었습니다

저는 정리를 잘 못합니다. 책상도, 폴더도, 머릿속도. 그래서 정리해주는 도구가 절실히 필요했고, 결국 직접 만들게 됐습니다. 정리에 가장 약한 사람이 만들었기에, 가장 정리를 못하는 사람의 마음을 압니다.

매일 아침 노트북을 켜면 같은 풍경이 펼쳐졌습니다. character_v01_final, character_v01_final_real, character_v02_finalfinal, 그리고 reference_2024-04-26-혹시이거였나. 어제 저장해 둔 보이스를 찾으려고 다섯 개의 폴더를 열었습니다. 그제 비교한 영상 두 개가 어떤 모델이었는지 기억나지 않았습니다. 좋은 결과가 나왔던 그 프롬프트는 분명히 한 시간 전에 적어두었는데, 어디로 사라졌는지 도무지 알 수 없었습니다.

만드는 일은 즐거웠습니다. 그런데 만들기 위해 지나가야 하는 잡일들은 즐겁지 않았습니다. ChatGPT에서 스토리를 쓰고, ElevenLabs에서 보이스를 만들고, Suno로 음악을 깔고, Runway에서 영상을 뽑고, 그 결과를 다시 다른 폴더에 옮겨 적는 일. 한 편의 영상을 위해 다섯 개의 탭과 일곱 개의 폴더를 오가며 흘려보낸 시간이, 정작 새 아이디어를 떠올리고 다듬는 시간보다 훨씬 길었습니다.

그러다 어느 새벽, 깨달았습니다. 제가 게으른 게 아니었구나. 도구가 정리를 도와주지 않아서였구나. 만들기와 정리가 같은 손에 맡겨져 있는 한, 만들기는 절반의 즐거움밖에 가질 수 없었구나. 만드는 사람이 정리까지 해야 한다면, 결국 만들지 못하게 되는구나.

"만들기는 사람이, 정리는 도구가. 그 분리가 작업을 다시 즐겁게 만든다."

그렇게 시작된 것이 soaviz studio입니다. 캐릭터 한 명을 만들면 그 캐릭터의 외형, 성격, 톤, 보이스, 참조 사진까지 한 곳에 묶입니다. 그 캐릭터가 등장하는 모든 샷에 자동으로 주입되니, 매번 같은 설명을 반복해 적지 않아도 됩니다. 생성한 결과물은 자동으로 라이브러리에 들어가고, 어떤 프롬프트로 어떤 모델에서 만들어졌는지가 영원히 기록됩니다. 마음에 드는 결과는 "채택", 그렇지 않은 결과는 그 이유와 함께 보관됩니다.

다음에 비슷한 작업을 할 때, Memory Graph가 비슷한 채택 샷을 자동으로 추천해 줍니다. 작업이 누적될수록 추천이 더 정확해지는 구조 — 어제의 작업이 오늘을 가볍게 만들고, 오늘의 결정이 내일의 시작을 빠르게 만듭니다. 시간이 지날수록 점점 더 빨라지는, 복리처럼 자라는 도구입니다.

아침마다 Today 페이지를 열면 어제 멈춘 자리에서 그대로 이어집니다. 캘린더에서 시작일과 종료일을 클릭하면 새 프로젝트가 자동으로 만들어지고, 그 안에 모든 결과가 알아서 정리됩니다. 폴더를 만들 일도, 이름을 지을 일도, 어디에 무엇이 있는지 헤맬 일도 없습니다. 한 편의 영상이 만들어지는 동안 사용자가 해야 하는 일은 단 하나, "만드는 일"뿐입니다.

한 화면 안에서 세계관을 쓰고, 캐릭터를 만들고, 보이스를 고르고, 스토리보드를 분해하고, 모델 A/B/C를 비교하고, 영상으로 뽑고, 자막을 붙이고, 마지막에 보관까지 — 새 탭을 열 일이 없습니다. 50가지 장르, 50가지 분위기, 50가지 카메라 스타일, 60가지 컬러 팔레트가 이미 준비되어 있어, 처음부터 다시 정리할 필요도 없습니다. 정리는 도구의 일이 됐고, 만드는 일은 사람의 일로 돌아왔습니다.

이 도구는 한 가지 단순한 원칙 위에 세워졌습니다. "한 번 만든 것은, 두 번 만들지 않는다." 한 번 등록한 캐릭터는 그 다음 모든 작업에 자동으로 합류합니다. 한 번 채택한 톤은 그 다음 모든 샷의 기본값이 됩니다. 한 번 좋은 결과가 나온 프롬프트는 자동으로 라이브러리에 보관되어, 비슷한 작업을 할 때 가장 먼저 떠오릅니다. 이것이 복리처럼 자라는 작업실의 의미입니다 — 처음에는 빈 도구였지만, 한 달 뒤에는 당신만의 시그니처를 기억하는 어시스턴트가 되어 있습니다.

정리에 약하다는 것이 부끄러움이 아니라는 사실도, 이 도구를 만들면서 배웠습니다. 정리에 약한 사람들은 보통 다른 일에 강합니다. 무언가를 새로 만들어내는 일, 아이디어를 떠올리는 일, 감정을 따라가는 일. 그런 사람들에게 정리를 강요하면, 그 강점들이 흐려집니다. 그래서 정리는 도구의 몫으로 옮겨야 합니다. 그래야 만드는 사람이 만드는 일에만 집중할 수 있습니다. 도구가 정리를 책임지면, 사람은 더 자유로워집니다.

저는 매일매일 시간을 녹여 이 도구를 만들고 있습니다. 새벽까지 다듬고, 아침에 깨어 다시 다듬습니다. 그렇게 흘려보낸 저의 수많은 시간이 모여, 여러분께는 커피 한 잔의 여유로 돌아가기를 바랍니다.

폴더를 헤매던 30분이, 따뜻한 아메리카노 한 잔의 시간으로. 잃어버린 프롬프트를 찾던 한 시간이, 좋아하는 음악을 듣는 짧은 산책으로. 정리해야 한다는 부담이 있던 자리에, 다시 만들고 싶은 마음이 들어서기를. 작업자의 관점에서 만들어진 도구는, 결국 작업자의 시간을 돌려주는 도구여야 한다고 믿습니다. 작업의 결과만큼이나, 그 과정에서 흘러간 시간도 소중하기 때문입니다.

아직 부족한 부분이 많습니다. 만들고 있는 도중이고, 배우고 있는 도중입니다. 정리에 약한 사람이 만든 정리 도구이니, 어딘가는 분명히 어색할 겁니다. 사용해 보신 분들의 한 줄 한 줄이 다음 업데이트의 방향이 됩니다. 작은 불편함도, "이런 게 있었으면" 하는 작은 바람도, 들려주시면 그대로 반영하겠습니다. 크리에이터의 시간이 가장 비싼 자원이라는 단 하나의 믿음에서 시작한 도구이니, 그 시간이 정말로 가벼워졌는지 — 끝까지 함께 확인해 가고 싶습니다.

영상을 만드는 모든 분께, 그리고 만들고 싶지만 정리에 지쳐 자꾸 미루게 되는 모든 분께. 흩어진 폴더와 사라진 프롬프트와 비슷한 실수의 반복 — 그 부담을 이제는 도구가 대신 짊어지기를. 그래서 만드는 일이 다시 즐거워지기를. soaviz studio가 그 작은 한 걸음을 도울 수 있다면, 그 다음 걸음은 만드는 분들의 몫으로 남기고 싶습니다.

당신의 시간을 돌려드리는 일, 거기까지가 우리의 일입니다.

3분 만에 시작하기

가입 → 첫 프로젝트 → 첫 결과물. 진짜 끝.

  1. 회원가입 웹 브라우저에서 app.soaviz.studio로 들어가세요. Google 계정이 있으면 1초 만에 가입됩니다. 이메일·비밀번호로도 가입 가능해요.
  2. 요금제 선택 (선택) 가입 즉시 무료 200 크레딧이 자동 지급됩니다. 모든 페이지를 둘러보고 기본 생성 흐름을 바로 테스트할 수 있어요. 고급 모델과 API Router는 유료 플랜에서 열립니다. 자세한 안내는 AI 사용량 안내 →
  3. 첫 프로젝트 만들기 오른쪽 캘린더에서 시작일과 종료일을 차례로 클릭하면 자동으로 새 프로젝트 창이 뜹니다. 이름만 입력하면 끝.
  4. 도구 골라서 만들어 보기 왼쪽 메뉴에서 만들고 싶은 도구를 선택하세요. 보이스 → 글 입력 → 생성 / 영상 → 프롬프트 입력 → 생성. 결과물은 자동으로 보관됩니다.
팁 — 시작 화면(Today)에는 진행 중인 프로젝트가 5개씩 페이지로 보입니다. 무료 플랜은 5개까지, Pro는 30개까지 만들 수 있고, 완성한 프로젝트는 ✓ 버튼으로 보관함에 정리할 수 있어요.

첫 영상 만들어 보기

5분 시나리오 — 캐릭터 1명 + 보이스 1줄 + 영상 1컷.

  1. 캐릭터 만들기 왼쪽 메뉴 → 캐릭터 만들기 → 새 캐릭터. 직접 입력하거나 🎲 주사위 버튼으로 랜덤 자동 생성. 이름·성격·외형이 한 번에 완성됩니다.
  2. 스토리 쓰기 캐릭터 만들기 페이지에서 세계관과 줄거리를 입력하고 ✨ 스토리 자동 생성을 누르세요. 길이는 짧음 / 보통 / 김 / 매우 김 등 6단계로 선택할 수 있어요.
  3. 목소리로 읽어 주기 보이스 (TTS) → 글 입력 → 보이스 선택 → 생성. 결과물은 화면 위쪽 미니 플레이어에서 바로 들어볼 수 있습니다.
  4. 영상 만들기 영상 생성 → 모델 선택(예: Veo 3, Hunyuan 등) → 원하는 장면 묘사 → 생성. 결과는 자동으로 보관됩니다.
  5. 완성! 정리하기 작업이 끝났으면 오늘(Today) → 해당 프로젝트의 ✓ 완료 처리 버튼을 누르세요. 완료된 작업은 보관함에 정리되고, 진행 중 목록은 깔끔해집니다.

알아두면 좋은 6가지 용어

처음 보는 단어가 많죠? 한 번만 보면 됩니다.

1. 프로젝트

한 편의 작품 단위입니다. 시작일·마감일·색상을 정해두면 진행 상황을 한눈에 볼 수 있어요.

2. 스토리보드 (Pipeline)

한 프로젝트 안의 장면들을 정리하는 곳. 작품 → 에피소드 → 장면 → 컷 4단계로 구성됩니다. 글로 쓴 스토리를 한 번에 장면들로 나눠주는 기능도 있어요.

3. 캐릭터 / 룩북 / 영감

여러 작품에서 재사용할 수 있는 자료들입니다:

  • 캐릭터 — 등장인물 (외형·성격·목소리·역할)
  • 룩북 — 작품의 분위기 (색감·조명·스타일)
  • 영감 라이브러리 — 참고할 영상들 모음 (장면별 타임코드 표시)

4. 결과물 & 작업 일기

만든 보이스·이미지·영상·텍스트는 모두 자동으로 보관됩니다. 언제 무엇을 만들었는지도 자동으로 기록되어 나중에 회고할 때 유용해요.

5. 안전 보관함 (Vault)

본인 AI 서비스 키처럼 민감한 정보는 강력한 암호화로 보호됩니다. soaviz studio 직원도 평문으로 볼 수 없어요.

Decision Library

모든 결과물은 의사결정의 흔적입니다. 채택·후보·탈락을 기록해 다음 작업의 자산으로 만드세요.

왜 이게 중요한가요?

AI로 작품을 만들다 보면 같은 장면을 여러 번 시도하게 됩니다. 어떤 건 채택하고, 어떤 건 탈락시키죠. 그 의사결정의 이유가 쌓이면 다음 작업이 빨라집니다 — "지난번에 이런 분위기는 캐릭터가 어색했지" 같은 직관이 데이터로 남거든요.

4가지 상태

상태의미
○ 후보방금 만든 결과물. 아직 채택 결정 전
✓ 채택본편에 사용할 베스트 — 그린 배지
✕ 탈락이번엔 안 쓸 결과 — 사유와 함께 보관
📦 보관당장은 안 쓰지만 나중을 위해 남겨두는 자료

탈락 사유 7가지

아이콘사유언제 쓰나요?
🎨분위기 안 맞음전반적인 톤·무드가 작품과 어긋남
👤캐릭터 다름주인공의 외모·표정이 다르게 나옴
🎬구도 별로카메라 앵글·구도가 의도와 다름
✨디테일 흐림화질·해상도·디테일이 부족
💸비용 비효율품질 대비 생성 비용이 높음
🔁다른 안이 좋음비교 대상 중 더 나은 게 있음 (Compare 자동)
📌기타위에 해당하지 않는 사유 (메모로 보충)

사용법

  1. 왼쪽 메뉴 → Decision Library 모든 결과물이 한 곳에 모입니다. 상단 탭으로 상태별 필터링.
  2. 마우스 올리면 빠른 액션 카드 우상단에 ✓ ✕ 📦 버튼이 나타납니다. ✓ 한 번에 채택, ✕ 누르면 사유 선택 모달.
  3. Compare Mode가 가장 빠릅니다 비교 모드에서 베스트 한 번 클릭하면 → 자동으로 채택 + 나머지 후보들은 "다른 안이 좋음(🔁)" 사유로 자동 탈락 처리.
  4. 나중에 회고할 때 보세요 한 달 뒤 "탈락" 탭만 보면 사유별 패턴이 보입니다 — 자주 "캐릭터 다름"이 뜬다면 캐릭터 묘사 프롬프트를 더 다듬어야 한다는 신호.
마찰 0 설계 — 일반 생성에서는 자동으로 "후보"로 들어가고, 사유 입력은 비교 모드 또는 명시적 ✕ 클릭 때만 묻습니다. 매번 답하지 않아도 됩니다.

곧 추가될 기능

  • "이거랑 비슷한" 검색 — 마음에 드는 결과의 분위기와 닮은 과거 작업을 자동으로 찾아줍니다 (3개월 안)
  • Reference Graph — 내 작업물의 시그니처 패턴을 시각화 (3개월 안)
  • 탈락 사유 통계 — 가장 많이 탈락하는 이유 분석 → 프롬프트 개선 힌트

AI 사용량 안내

작업 화면은 모두 열려 있고, 실제 AI 실행 시 크레딧과 모델 권한을 확인합니다.

두 가지 사용 방식 — 기본 생성은 soaviz 크레딧으로 실행합니다. Pro 이상에서는 API Router/BYOK를 켜서 OpenAI·Anthropic·ElevenLabs·Replicate 등 본인 API 키를 연결하고 고급 모델 비용을 직접 관리할 수 있습니다.

크레딧이 뭔가요?

크레딧은 AI 사용량을 측정하는 단위입니다. 글 한 줄 생성, 보이스 한 문장, 영상 한 컷마다 일정 크레딧이 차감되며, 매월 사용할 수 있는 크레딧이 플랜마다 정해져 있습니다.

AI 서비스 키를 따로 등록해야 하나요?

기본 사용에는 필요 없습니다. 무료/Standard는 soaviz 크레딧으로 기본·Premium 모델을 실행합니다. Pro 이상에서는 API Router/BYOK로 본인 API 키를 등록해 고급 모델과 비용 로그를 직접 관리할 수 있습니다.

크레딧은 어떻게 사용되나요?

스토리 생성·보이스·음악·영상 등 AI 작업마다 일정 크레딧이 차감되며, 정확한 단가는 작업 시 미리 표시됩니다. 자세한 단가표는 AI 사용량 안내 →

크레딧이 부족하면 어떻게 되나요?

월 크레딧을 모두 소진하면 다음 결제 주기까지 AI 생성이 일시 중단됩니다. 기존 작업물 편집·내보내기·검토는 계속 가능합니다. 대량 작업이 필요하다면 한 단계 위 플랜으로 업그레이드를 권장합니다(Standard 5,000 → Pro 50,000 크레딧).

스마트폰에서도 사용 가능한가요?

가능하지만 화면이 작아 둘러보기·간단한 편집 정도만 추천합니다. 본격적인 작업은 노트북·데스크톱 (Chrome 또는 Safari 최신 버전)을 권장합니다.

인터넷 없이도 작업이 되나요?

이미 만들어 둔 글·스토리·캐릭터 편집은 가능합니다. 인터넷이 다시 연결되면 자동으로 저장돼요. 단 AI 생성(보이스·영상 등)은 인터넷이 필요합니다.

저작권은 누구에게 있나요?

사용자가 만든 모든 콘텐츠의 저작권은 사용자에게 있습니다. soaviz studio는 서비스 제공에 필요한 최소한의 권리만 가집니다. 자세한 내용은 이용약관 →

AI가 만든 결과물을 광고·제품에 사용해도 되나요?

각 AI 서비스(ChatGPT·ElevenLabs 등)의 약관에 따릅니다. 대부분 상업적 사용이 가능하지만, 서비스마다 다르니 직접 확인이 필요합니다.

다른 영상 편집 프로그램으로 가져갈 수 있나요?

네. DaVinci Resolve, Final Cut Pro, Premiere Pro 등 주요 영상 편집 프로그램으로 가져가실 수 있습니다.

해외에서도 사용 가능한가요?

전 세계 어디서나 사용 가능합니다. 한국 외 지역 사용자에게는 미국 데이터센터 옵션도 제공해 더 빠른 속도를 보장합니다.

  • 크레딧 잔량 — 환경설정 → Billing에서 이번 달 잔여 크레딧 확인
  • 인터넷 연결 — Wi-Fi/데이터가 끊겨 있지 않은지 확인
  • 모델 점검 안내 — 일부 외부 AI 서비스가 일시적으로 점검 중일 수 있음 (잠시 후 재시도)
  • 그래도 안 되면 support@soaviz.studio

"크레딧 부족" 메시지가 나와요

이번 달 사용 가능한 크레딧을 모두 소진한 경우입니다.

  • 다음 달까지 대기 — 결제 주기 시작일에 자동 충전
  • 플랜 업그레이드 — Standard(5,000) / Pro(50,000)로 즉시 확장
  • 이미 만든 결과물은 계속 편집·내보내기 가능

로그인이 안 돼요

  • 로그인 화면 → "비밀번호 잊으셨나요?" → 이메일로 재설정
  • 2단계 인증 코드를 잃어버렸다면 가입 시 받은 백업 코드 사용
  • 그래도 안 되면 support@soaviz.studio

결제가 자꾸 실패해요

국내 카드라면 카카오페이·네이버페이·토스페이가 가장 안정적입니다. 해외 카드는 카드사에서 해외 결제 차단을 풀어주셔야 해요.

영상 만드는 게 너무 오래 걸려요

모델별 속도 차이입니다:

  • 빠른 변환 (Veo 3 Fast) — 1~2분
  • 고품질 (Veo 3) — 5~10분
  • Hunyuan — 3~5분

먼저 빠른 모델로 테스트하고, 마음에 드는 결과를 고품질로 다시 만드는 것을 추천합니다.

화면이 멈추거나 느려져요

Chrome 또는 Safari 최신 버전을 사용하세요. 다른 탭이 너무 많으면 닫으시고, 환경설정 → 저장소 → 임시 캐시 정리를 눌러 보세요.

"soaviz studio"는 (주)소아비즈가 제공하는 AI 영상 제작 플랫폼입니다.

저작권

사용자가 만든 모든 콘텐츠는 사용자 본인의 것입니다. soaviz studio는 서비스 제공에 필요한 최소한의 권리(저장·동기화 등)만 가집니다.

금지 사항

  • 음란물·아동 착취물·폭력 콘텐츠 만들기
  • 딥페이크 등 다른 사람의 명예를 훼손하는 콘텐츠
  • 서비스를 무단으로 침입·해킹하려는 시도
  • 다른 사용자에 대한 괴롭힘·혐오 표현

  • 외부 AI 서비스(ChatGPT·ElevenLabs 등)의 장애로 인한 손해
  • 본인 부주의로 인한 데이터 손실 (예: 백업 미실시 후 계정 삭제)
