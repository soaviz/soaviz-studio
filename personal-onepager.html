The Problem

혼자 영상 만드는데
창은 10개씩 띄워야 합니다.

Kling 탭 · Runway 탭 · ChatGPT 탭 · ElevenLabs 탭 · 메모장 · Figma · 폴더 정리… 혼자 만드는 사람일수록 도구 사이를 가장 많이 오갑니다. 정리할 사람이 본인뿐이라.

Kling · Runway · Sora
영상은 쌓이는데 맥락은 사라집니다
v1·v2·v3 시도는 늘어나는데, 어느 프롬프트가 어느 결과를 만들었는지 며칠 지나면 잊어버립니다.
ChatGPT · Claude
스토리는 흩어집니다
캐릭터 설정·세계관·샷리스트가 각각 다른 대화창에. 한 번 잃으면 처음부터 다시 만들어야 합니다.
ElevenLabs · Suno
사운드도 따로 작업합니다
내레이션·BGM·SFX가 다른 도구. 영상에 맞추려면 결국 다시 다운받고 영상 편집기에서 끼워 넣어야 합니다.
엑셀 · 폴더 · Notion
관리는 본인 몫
외주 받으면 클라한테 "어떤 모델 썼나요?" 질문이 오지만, 정리해놓은 게 없어서 매번 추적해야 합니다.
결과: 1인 크리에이터가 도구 10개를 혼자 관리합니다.
창작 시간보다 정리·이동·복사에 더 많은 시간을 씁니다.

How it works

기획부터 납품까지, 4단계.

STEP 1
아이디어 → 스토리
한 줄 컨셉을 입력하면 AI가 시놉시스·캐릭터·세계관까지 자동 정리.
STEP 2
샷 분해 + 생성
스토리가 자동으로 샷 단위로 쪼개집니다. 크레딧으로 Kling·Runway·Sora 즉시 호출.
STEP 3
버전 비교 + 채택
한 샷에 여러 시도 누적. 좋은 버전만 별 표시 → 자동 라이브러리 저장.
STEP 4
편집 → Export
편집 타임라인 + 보이스·BGM 합치기 → MP4로 한 번에 Export. 이때까지 한 화면.

FAQ

자주 묻는 질문

API 키를 등록해야 하나요?
아니요. 개인 플랜은 API 키 없이 SOAVIZ 크레딧으로 모든 제작이 실행됩니다. 키 발급, 프로바이더 결제 관리, 보안 설정이 전혀 필요하지 않습니다. 자체 AI 인프라 연결(BYOK)은 Team OS에서 기관 관리자에게만 제공됩니다.
어떤 AI 모델을 쓸 수 있나요?
Kling 1.6, Runway Gen-3, OpenAI Sora, ElevenLabs, Suno, Stability AI 등 주요 영상·음성·음악 생성 엔진을 지원합니다. 모두 SOAVIZ 크레딧으로 호출합니다 — 키 발급이나 별도 결제 관리가 필요 없습니다.
외주 받았을 때 클라이언트한테 어떻게 보여주나요?
Creator 플랜은 영상 + 캡션을 Export. Studio로 한 클릭 전환하면 클라이언트 리뷰 링크 + 권리 리포트까지 자동. 외주가 커질 때 별도 도구 갈아타지 않고 그대로 확장됩니다. 데이터는 그대로 남아있어요.
기존 ChatGPT·Kling·Runway 작업을 가져올 수 있나요?
프롬프트는 복붙으로 즉시 옮길 수 있습니다. 영상 파일은 라이브러리에 업로드하면 샷에 연결됩니다. 캐릭터·세계관 설정은 Story Bible에 입력하면 모든 새 생성에 자동 주입됩니다.
Rights Report는 법적 효력이 있나요?
Rights Report는 SOAVIZ가 사용된 모델·라이선스·프롬프트·승인 흐름을 자동으로 기록한 문서입니다. 법적 효력은 관할권에 따라 다르지만, 클라이언트·법무팀·감사 대응 시 사실 기반 자료로 사용할 수 있습니다. Business 플랜 이상에서 PDF로 export 가능합니다.
베타에 참여하면 어떤 혜택이 있나요?
베타 사용자는 정식 출시 후 6개월간 Studio 플랜을 50% 할인된 가격으로 사용할 수 있습니다. 제품 로드맵에 직접 영향을 줄 수 있는 정기 라운드테이블에도 초대됩니다.
데이터는 어디에 저장되나요?
프로젝트 메타데이터는 SOAVIZ의 보안 인프라에, 영상 파일은 사용자가 지정한 스토리지(자체 S3, GCS 등)에 저장 가능합니다. Enterprise 플랜은 전용 인스턴스 옵션을 제공합니다.
