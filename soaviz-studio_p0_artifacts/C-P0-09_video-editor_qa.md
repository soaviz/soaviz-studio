# C-P0-09: video-editor QA 체크리스트

video-editor 페이지(48.8 KB로 이미 구현됨)의 실제 작동 검증.

## 진입 검증

- [ ] 사이드바에서 "Edit · Video Editor" 클릭 → 정상 진입
- [ ] 게스트 상태에서 진입 시도 → 로그인 게이트 노출
- [ ] 로그인 후 진입 → 대시보드 정상 표시

## 컴포넌트 검증

- [ ] **작업 캘린더 위젯** — 데이터 표시 (오늘 일정 N건)
- [ ] **워크스페이스 패널** — ProjectProgressPanel 5단계 노출
- [ ] **프로젝트 진행률 바** — 0-100% 정확히 계산
- [ ] **"다음 단계" 버튼** — 클릭 시 다음 페이지로 자동 이동
- [ ] **"이어서 작업" 버튼** — 마지막 작업 위치로 복귀

## 데이터 영속화 검증

- [ ] 페이지 새로고침 후 → 진행률 유지
- [ ] 다른 페이지 갔다가 복귀 → 진행률 유지
- [ ] localStorage/Dexie 데이터 정상 (개발자 도구 → Application 탭)
